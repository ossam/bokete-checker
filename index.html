⚙️ 初期設定

Google Apps Script のWebアプリURLを入力してください。

📖 URLの取得方法（クリックで開く）
  1. 📊 スプレッドシートを開く
  2. メニュー「拡張機能」→「Apps Script」
  3. 📄 Code.gs を開く → 「Raw」→ 全選択コピー → エディタに貼り付けて Ctrl+S で保存
  4. 右上「デプロイ」→「新しいデプロイ」
  5. ⚙️アイコン → 種類：ウェブアプリ
  6. 実行するユーザー：自分 ／ アクセス：全員
  7. 「デプロイ」→ 表示されたURLをコピーして下の欄に貼り付け
⚠️ コード変更時は「デプロイを管理」ではなく必ず「新しいデプロイ」を選んでください。URLが毎回変わります。

❓ 使い方ガイド

⭐ 評価する

  • 各カードの★をクリックで評価（1〜3つ星）
  • 同じ★をもう一度クリックで取り消し
  • 評価はスプレッドシートに即時反映されます

🔍 フィルタ・ソート

  • 上部タブで評価別に絞り込み（全て / 未評価 / ★ / ★★ / ★★★）
  • 「新しい順」「古い順」で並び替え
  • 50件ずつ表示、「もっと見る」で追加読み込み

🖼️ 画像について

  • bokete.jpのCDN制限により、一部の画像は表示されません
  • 「🖼️ boketeで見る ↗」リンクをクリックするとbokete.jpで確認できます

🛠️ セットアップ手順

  1. スプレッドシートの「拡張機能」→「Apps Script」を開く
  2. エディタに gas/Code.gs の内容を貼り付けて保存
  3. 「デプロイ」→「新しいデプロイ」→ ウェブアプリ → アクセス「全員」
  4. 発行されたURLを⚙️ボタンから入力

📷 新しいボケを追加した時

  1. スプレッドシートに新しい行を追加
  2. GASエディタで batchFetchImageUrls を▶️実行
  3. このアプリで「↻ 再読込」を押す
